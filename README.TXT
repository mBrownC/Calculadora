# Calculadora Simple

Este es un proyecto de una calculadora simple desarrollada utilizando HTML, CSS y JavaScript.

## Características

- Suma, resta, multiplicación y división de números.
- Interfaz simple y fácil de usar.

## Instalación

No se requiere ninguna instalación especial. Solo descarga los archivos y ábrelos en un navegador web compatible.

## Uso

1. Abre el archivo `index.html` en tu navegador web.
2. Utiliza los botones numéricos para ingresar los números.
3. Utiliza los botones de operación (+, -, *, /) para realizar las operaciones.
4. El botón "C" limpia la pantalla y reinicia la calculadora.

## Personalización

Si deseas personalizar la apariencia de la calculadora, puedes modificar el archivo CSS (`styles.css`) para ajustar los colores, tamaños y estilos según tus preferencias.
